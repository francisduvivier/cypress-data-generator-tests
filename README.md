# Cypress Test Suite for the Test Data Generator

Random data generator test suite for https://bignited.be/services/test-data-generator/

## How to use

### Running tests in headless mode

`yarn cypress:run`

### Running test with cypress ui

`yarn cypress:open`

## How to develop
### Git hook setup
`yarn prepare`
### Linting Check

`yarn lint`

### Fix lint issues
`yarn lint:fix`
